# Activate Sagaz Ecosystem

## Invocacao Curta

Agora o Sagaz tambem existe como skill local do Codex em:

`C:\Users\YOUR_USER\.codex\skills\sagaz`

Use simplesmente:

```text
Sagaz: [sua tarefa]
```

Ou:

```text
Use o Sagaz para [sua tarefa]
```

Copie a intencao abaixo para iniciar qualquer trabalho com este ecossistema:

```text
Ative o Sagaz AI Orchestration Ecosystem.
Leia somente ai-orchestration-ecosystem/quickstart.md primeiro.
Escolha o menor modo e o menor squad suficientes.
Use especificacao detalhada quando o risco justificar.
Valide a entrega com quality gates e testing matrix.
Consuma poucos tokens no processo e salve artefatos importantes em Markdown.
```

## Atalhos

### Criar produto novo

```text
Ative o Sagaz AI Orchestration Ecosystem em modo factory com o squad product-factory.
```

### Revisar codigo

```text
Ative o Sagaz AI Orchestration Ecosystem em modo audit com o squad code-audit.
```

### Refatorar com seguranca

```text
Ative o Sagaz AI Orchestration Ecosystem em modo refactor com o squad refactor-lab.
```

### Manter projeto existente

```text
Ative o Sagaz AI Orchestration Ecosystem em modo squad com o squad maintenance-ops.
```

### Operar GitHub

```text
Sagaz: use github-ops para criar branch, commit, push e pull request com testes e revisao de seguranca.
```

### GitHub Guiado

```text
Sagaz: cuide do Git e GitHub por mim. Avise quando uma acao for necessaria, explique o motivo em linguagem simples e peca minha confirmacao antes de executar.
```

### Proatividade Global

```text
Sagaz: aja proativamente em todo o projeto. Sugira proximos passos necessarios ou recomendaveis, explique motivo/impacto/risco e peca permissao antes de executar acoes relevantes.
```

### Design De Alto Padrao

```text
Sagaz: use design-studio para criar uma interface com UX/UI premium, design system completo, responsividade, acessibilidade e validacao visual.
```

### Recomendacao De Stack

```text
Sagaz: recomende a melhor stack para esta aplicacao, explicando custo, velocidade, escalabilidade, manutencao, deploy e facilidade de futuras alteracoes.
```

### Pipeline Com Handoffs

```text
Sagaz: siga um pipeline sequencial com handoff entre equipes. Ao final de cada fase, explique o que foi feito, o que a proxima equipe fara e peca minha permissao antes de avancar.
```

### Workflows Prontos

```text
Sagaz: use o workflow adequado para este projeto e mantenha um run-state em Markdown com fase atual, decisoes, aprovacoes, handoffs, testes e riscos.
```

### Mobile App

```text
Sagaz: use mobile-app-studio para criar um app Android/iOS com stack recomendada, UX mobile, testes, build e checklist de release.
```

## Primeira Pergunta Recomendada

Quando a tarefa for ampla, pergunte:

```text
Qual e o resultado final verificavel que precisa existir no fim desta tarefa?
```
